# Copilot Studio Solution

## DynamicsReleasePlannerHelperAgent_1_0_0_1.zip

This is the Copilot Studio agent solution for the Release Planner Helper Agent.

### Import Instructions

1. Go to [make.powerapps.com](https://make.powerapps.com) → **Solutions** → **Import**
   - Or open [Copilot Studio](https://copilotstudio.microsoft.com) → **Import Agent**
2. Select `DynamicsReleasePlannerHelperAgent_1_0_0_1.zip`
3. Follow the wizard to create/sign in to required connections
4. Click **Import** and wait for completion
5. After import, open the agent in Copilot Studio and configure the MCP server connection

> **Note:** The agent requires a running MCP server instance. See the [MCP Server Guide](../MCP_SERVER_GUIDE.md) for setup instructions.

### Prerequisites

- Microsoft Copilot Studio access
- Copilot Studio Credits Capacity
- A deployed MCP server instance (local or Azure)
